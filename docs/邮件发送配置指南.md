## 概述

`email-sender` skill 允许你通过 SMTP 协议发送邮件，支持：

- ✅ 纯文本和 HTML 邮件
- ✅ 附件发送
- ✅ 抄送 (CC) 和密送 (BCC)
- ✅ 多收件人
- ✅ 自定义 SMTP 服务（QQ 邮箱、Gmail、公司邮箱等）

## 配置步骤

### 步骤 1：获取 SMTP 授权码

#### QQ 邮箱

1. 登录 QQ 邮箱网页版
2. 进入「设置」→「账户」
3. 开启「POP3/SMTP 服务」
4. 生成授权码（不是登录密码！）

#### Gmail

1. 登录 Gmail
2. 开启两步验证
3. 生成应用专用密码
4. 使用该密码作为 SMTP 密码

#### 公司邮箱

联系 IT 部门获取：
- SMTP 服务器地址
- 端口号
- 用户名和密码

### 步骤 2：创建配置文件

创建 `.env` 文件或 `config.json`：

#### 方式 A：使用 .env 文件

```bash
# 在 skills/email-sender 目录下创建 .env 文件
cat > .env << EOF
SMTP_SERVER=smtp.qq.com
SMTP_PORT=465
SMTP_USER=your_email@qq.com
SMTP_PASSWORD=your_auth_code
SMTP_USE_SSL=true
EOF
```

#### 方式 B：使用 config.json

```json
{
  "email": {
    "smtp_server": "smtp.qq.com",
    "smtp_port": 465,
    "smtp_user": "your_email@qq.com",
    "smtp_password": "your_auth_code",
    "use_ssl": true,
    "use_tls": false
  }
}
```

### 步骤 3：测试连接

```bash
# 测试 SMTP 连接
python -c "
import smtplib
from email.mime.text import MIMEText

# 配置
smtp_server = 'smtp.qq.com'
smtp_port = 465
username = 'your_email@qq.com'
password = 'your_auth_code'

# 创建连接
server = smtplib.SMTP_SSL(smtp_server, smtp_port)
server.login(username, password)
print('✅ SMTP 连接成功！')
server.quit()
"
```

## 使用示例

### 示例 1：发送简单文本邮件

```python
from email_sender import send_email

send_email(
    to=['recipient@example.com'],
    subject='测试邮件',
    body='这是一封测试邮件的内容。'
)
```

### 示例 2：发送 HTML 邮件

```python
from email_sender import send_email

html_content = """
<html>
<body>
    <h1>欢迎</h1>
    <p>这是一封 <strong>HTML</strong> 邮件。</p>
    <ul>
        <li>项目 1</li>
        <li>项目 2</li>
    </ul>
</body>
</html>
"""

send_email(
    to=['recipient@example.com'],
    subject='HTML 测试邮件',
    body=html_content,
    html=True
)
```

### 示例 3：带附件的邮件

```python
from email_sender import send_email

send_email(
    to=['recipient@example.com'],
    subject='带附件的邮件',
    body='请查收附件。',
    attachments=['./docs/report.pdf', './data.xlsx']
)
```

### 示例 4：抄送和密送

```python
from email_sender import send_email

send_email(
    to=['primary@example.com'],
    cc=['manager@example.com'],
    bcc=['archive@example.com'],
    subject='项目更新',
    body='项目进度更新...'
)
```

### 示例 5：完整示例（命令行）

```bash
# 使用命令行发送
python -m skills.email-sender \
  --config config.json \
  --to recipient@example.com \
  --cc manager@example.com \
  --subject "项目周报 - 2026-03-14" \
  --body "本周工作完成情况如下..." \
  --attachment "./reports/weekly_report.pdf" \
  --html
```

### 示例 6：批量发送

```python
from email_sender import send_email

recipients = [
    'user1@example.com',
    'user2@example.com',
    'user3@example.com'
]

for recipient in recipients:
    send_email(
        to=[recipient],
        subject='个性化通知',
        body=f'尊敬的 {recipient}，...'
    )
```

## 常见问题

### Q1: 认证失败

**问题：** `SMTPAuthenticationError`

**解决方案：**
- 确认使用的是授权码，不是登录密码
- 检查 SMTP 服务是否已开启
- 验证用户名和密码是否正确

### Q2: 连接超时

**问题：** `Connection timed out`

**解决方案：**
- 检查网络连接
- 确认 SMTP 服务器地址和端口正确
- 检查防火墙设置
- 尝试切换 SSL/TLS 模式

### Q3: 附件过大

**问题：** 邮件发送失败

**解决方案：**
- 大多数邮件服务器限制附件大小为 25MB
- 使用云存储链接代替大附件
- 压缩文件后发送

### Q4: 邮件进入垃圾箱

**问题：** 收件人反馈邮件进入垃圾箱

**解决方案：**
- 使用公司域名邮箱
- 配置 SPF、DKIM 记录
- 避免使用敏感词汇
- 控制发送频率

## 安全建议

### ⚠️ 重要安全提示

1. **永远不要硬编码密码**
   ```python
   # ❌ 错误做法
   password = "my_secret_password"
   
   # ✅ 正确做法
   import os
   password = os.getenv('SMTP_PASSWORD')
   ```

2. **使用环境变量存储敏感信息**
   ```bash
   # .env 文件（加入 .gitignore）
   SMTP_PASSWORD=your_auth_code
   ```

3. **限制文件权限**
   ```bash
   chmod 600 .env config.json
   ```

4. **定期更换授权码**
   - 每 3-6 个月更换一次
   - 离职员工立即撤销权限

5. **使用专用邮箱**
   - 为自动发送创建专用邮箱
   - 不要使用个人主邮箱

### .gitignore 配置

确保敏感文件不被提交：

```gitignore
# 邮件配置
.env
config.json
*.key
*.pem

# 凭据文件
credentials/
secrets/
```

## 最佳实践

### 邮件模板

创建可重用的邮件模板：

```python
# templates/email_templates.py

WEEKLY_REPORT_TEMPLATE = """
<html>
<body>
    <h2>周报 - {week}</h2>
    <p>负责人：{author}</p>
    
    <h3>本周完成</h3>
    {completed_items}
    
    <h3>下周计划</h3>
    {planned_items}
</body>
</html>
"""

def format_weekly_report(week, author, completed, planned):
    return WEEKLY_REPORT_TEMPLATE.format(
        week=week,
        author=author,
        completed_items=completed,
        planned_items=planned
    )
```

### 错误处理

```python
from email_sender import send_email, EmailSendError

try:
    send_email(
        to=['recipient@example.com'],
        subject='重要通知',
        body='内容...'
    )
    print('✅ 邮件发送成功')
except EmailSendError as e:
    print(f'❌ 邮件发送失败：{e}')
    # 记录日志或重试
```

### 发送日志

```python
import logging
from datetime import datetime

logging.basicConfig(
    filename='email_log.txt',
    level=logging.INFO,
    format='%(asctime)s - %(levelname)s - %(message)s'
)

def send_with_logging(**kwargs):
    try:
        send_email(**kwargs)
        logging.info(f"邮件发送成功：{kwargs['to']}")
    except Exception as e:
        logging.error(f"邮件发送失败：{kwargs['to']} - {e}")
        raise
```
