# RAG 本地模型配置指南

## 问题

RAG 记忆系统默认从 HuggingFace 下载向量模型，如果网络不通会导致初始化失败：

```
Load pretrained SentenceTransformer: sentence-transformers/all-MiniLM-L6-v2
connect_tcp.failed host='huggingface.co'  ← 连接超时
RAG 记忆：✗
```

## 解决方案

### 方案 A：使用国内镜像源（推荐）

**步骤 1：设置环境变量**

```bash
# 临时设置（当前会话有效）
export HF_ENDPOINT=https://hf-mirror.com

# 永久设置（推荐）
echo 'export HF_ENDPOINT=https://hf-mirror.com' >> ~/.bashrc
source ~/.bashrc
```

**步骤 2：重启 pyclaw**

```bash
# 停止当前进程
pkill -f "pyclaw.*main.py"

# 重新启动
cd /home/zjm/.openclaw/workspace/pyclaw
python3 main.py --port 18790 --multi-agent
```

**效果：** 首次启动时会自动从镜像源下载模型（约 90MB），之后会使用缓存。

---

### 方案 B：手动下载模型到本地

**步骤 1：下载模型**

在有网络的环境下执行：

```bash
# 创建目录
mkdir -p /home/zjm/.local/models/sentence-transformers

# 使用 git 下载
cd /home/zjm/.local/models/sentence-transformers
git lfs install
git clone https://hf-mirror.com/sentence-transformers/all-MiniLM-L6-v2.git
```

**步骤 2：修改配置**

编辑 `config.json`（或 `config.example.json`）：

```json
{
  "rag": {
    "embedding_model": "sentence-transformers/all-MiniLM-L6-v2",
    "embedding_model_path": "/home/zjm/.local/models/sentence-transformers/all-MiniLM-L6-v2",
    "enable_vector_search": true
  }
}
```

**步骤 3：重启 pyclaw**

---

### 方案 C：禁用向量搜索（仅使用 BM25）

如果不需要语义搜索，可以禁用向量搜索，只使用 BM25（基于关键词的搜索引擎）：

**修改配置：**

```json
{
  "rag": {
    "enable_vector_search": false,  // 禁用向量搜索
    "retrieval_method": "bm25"      // 只使用 BM25
  }
}
```

**效果：**
- ✅ 启动快，不需要下载模型
- ✅ 适合关键词搜索场景
- ❌ 无法理解语义（如"猫"和"猫咪"）

## 模型说明

### 默认模型：all-MiniLM-L6-v2

- **大小：** 约 90MB
- **维度：** 384
- **速度：** 快
- **精度：** 良好
- **适用：** 通用文本嵌入

### 可选替代模型

| 模型 | 大小 | 维度 | 速度 | 精度 | 适用场景 |
|------|------|------|------|------|---------|
| `all-MiniLM-L6-v2` | 90MB | 384 | ⚡⚡⚡ | ⭐⭐⭐ | 通用（默认） |
| `all-mpnet-base-v2` | 420MB | 768 | ⚡⚡ | ⭐⭐⭐⭐ | 高质量语义 |
| `paraphrase-multilingual` | 470MB | 768 | ⚡⚡ | ⭐⭐⭐⭐ | 多语言支持 |
| `bge-small-zh-v1.5` | 130MB | 512 | ⚡⚡⚡ | ⭐⭐⭐⭐ | 中文优化 |

**中文场景推荐：** `bge-small-zh-v1.5`（北京智源实验室，中文效果更好）

## 常见问题

### Q1: 下载很慢怎么办？

使用镜像源：
```bash
export HF_ENDPOINT=https://hf-mirror.com
```

### Q2: 模型文件在哪里？

默认缓存位置：
```
~/.cache/huggingface/hub/
```

### Q3: 如何查看已下载的模型？

```bash
ls -lh ~/.cache/huggingface/hub/
```

### Q4: 可以多个项目共享模型吗？

可以！设置共享缓存目录：
```bash
export HF_HOME=/shared/huggingface
```
