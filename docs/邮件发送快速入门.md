# 邮件发送快速入门

> 5 分钟快速配置并开始发送邮件

---

## 🚀 快速开始

### 步骤 1: 安装依赖

```bash
# 确保在项目根目录
cd /path/to/pyclaw

# 安装 email-sender 相关依赖（如果需要）
pip install aiosmtplib email-validator
```

### 步骤 2: 配置邮箱

创建配置文件：

```bash
# 复制示例配置
cp config/email_config.example.json config/email_config.json

# 编辑配置
vim config/email_config.json
```

修改以下内容：

```json
{
  "email": {
    "smtp_server": "smtp.qq.com",
    "smtp_port": 465,
    "smtp_user": "your_email@qq.com",
    "smtp_password": "YOUR_AUTH_CODE",
    "use_ssl": true
  }
}
```

> ⚠️ **重要：** `smtp_password` 是授权码，不是登录密码！

### 步骤 3: 获取授权码（以 QQ 邮箱为例）

1. 登录 [QQ 邮箱网页版](https://mail.qq.com)
2. 点击「设置」→「账户」
3. 找到「POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV 服务」
4. 开启「POP3/SMTP 服务」
5. 点击「生成授权码」
6. 按提示发送短信验证
7. 复制生成的授权码

### 步骤 4: 测试发送

```bash
# 测试邮件（模拟模式）
python tools/email_workflow.py alert \
  --to "your_email@qq.com" \
  --title "测试邮件" \
  --message "这是一封测试邮件" \
  --level info \
  --dry-run

# 实际发送
python tools/email_workflow.py alert \
  --to "your_email@qq.com" \
  --title "测试邮件" \
  --message "这是一封测试邮件" \
  --level info
```

## 🛠️ 进阶使用

### 使用 Python 代码发送

```python
from tools.email_workflow import EmailWorkflow

# 初始化
workflow = EmailWorkflow(config_path="config/email_config.json")

# 发送文档通知
workflow.send_doc_notification(
    doc_path="./docs/技术文档.md",
    to=["team@company.com"],
    doc_type="技术文档",
    author="张三"
)

# 发送周报
workflow.send_weekly_report(
    week="2026-W11",
    content="本周工作正常",
    to=["manager@company.com"],
    completed=["任务 1", "任务 2"],
    planned=["任务 3", "任务 4"]
)

# 发送告警
workflow.send_alert(
    title="系统异常",
    message="数据库连接失败",
    to=["oncall@company.com"],
    level="error"
)
```

### 自定义 HTML 模板

1. 创建模板文件：`docs/templates/custom.html`
2. 在配置文件中引用：

```json
{
  "templates": {
    "document_notification": {
      "subject": "新文档：{doc_title}",
      "body_template": "docs/templates/custom.html"
    }
  }
}
```

### 批量发送

```python
from tools.email_workflow import EmailWorkflow

workflow = EmailWorkflow()

recipients = [
    "user1@company.com",
    "user2@company.com",
    "user3@company.com"
]

for recipient in recipients:
    workflow.send_alert(
        title="月度报告",
        message=f"您的月度报告已生成",
        to=[recipient],
        level="info"
    )
```

## 🔧 故障排查

### 问题 1: 认证失败

**错误信息：** `SMTPAuthenticationError`

**解决方案：**
- 确认使用的是授权码，不是登录密码
- 检查 SMTP 服务是否已开启
- 验证邮箱账号是否正确

### 问题 2: 连接超时

**错误信息：** `Connection timed out`

**解决方案：**
```bash
# 测试网络连接
ping smtp.qq.com

# 测试端口
telnet smtp.qq.com 465

# 检查防火墙
sudo ufw status
```

### 问题 3: 配置文件不存在

**错误信息：** `FileNotFoundError: config/email_config.json`

**解决方案：**
```bash
# 创建配置文件
cp config/email_config.example.json config/email_config.json
```

{

## 🔐 安全提示

1. **不要将配置文件提交到 Git**
   ```bash
   # 添加到 .gitignore
   echo "config/email_config.json" >> .gitignore
   ```

2. **使用环境变量存储敏感信息**
   ```bash
   export SMTP_PASSWORD="your_auth_code"
   ```

3. **定期更换授权码**
   - 建议每 3-6 个月更换一次
   - 员工离职时立即撤销

4. **使用专用邮箱**
   - 为自动发送创建专用邮箱
   - 不要使用个人主邮箱
