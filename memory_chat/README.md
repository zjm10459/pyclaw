# PyClaw 记忆文件

## 📁 说明

本目录 (`memory_chat/`) 存放**每日记忆文件**。

**长期记忆位置：** `../长期记忆.md`

---

## 📂 文件结构

```
memory_chat/
├── README.md             # 本文件
├── 2026-03-12.md         # 3 月 12 日记忆
├── 2026-03-13.md         # 3 月 13 日记忆
└── 2026-03-14.md         # 今日记忆
```

---

## 📝 文件用途

### `YYYY-MM-DD.md` - 每日记忆

**用途：**
- 📝 记录当天发生的原始日志
- 💬 对话摘要
- 📅 临时笔记

**特点：**
- 每天一个文件
- 自动创建
- 按日期组织

### `../长期记忆.md` - 长期记忆

**用途：**
- 🧠 精选的长期记忆
- 💡 重要决策、用户偏好、学到的教训
- 🔄 跨会话持久化

---

## 🎯 使用方式

### 1. 自动保存（推荐）

对 PyClaw 说：
```
记住，我更喜欢使用中文回复
```

AI 会自动调用 `remember` 工具保存到 `../长期记忆.md`。

### 2. 手动编辑

```bash
# 编辑今日记忆
vim 2026-03-14.md

# 编辑长期记忆
vim ../长期记忆.md
```

### 3. 使用工具

```python
from tools.memory_tools import remember, append_to_daily_memory

# 保存到长期记忆
remember(
    content="用户偏好中文",
    category="preference",
    tags=["语言"]
)

# 添加到今日记忆
append_to_daily_memory(
    content="完成了记忆系统重构",
    title="系统改进"
)
```

---

## 📊 记忆流程

```
用户说"记住这个"
    ↓
AI 调用 remember 工具
    ↓
保存到 ../长期记忆.md ✅
    ↓
下次会话读取
    ↓
AI 记住你的偏好 ✅
```

---

## ⚠️ 注意事项

1. **不要删除历史文件** - 包含重要记录
2. **定期回顾长期记忆** - 清理过时内容
3. **隐私注意** - 不要记录敏感信息
4. **文件格式** - 使用 Markdown 格式
